# Caesar Cipher
This is a simple web-based application used to encrypt text using the Caesar cipher technique. The application shifts letters forward in the alphabet based on a user-defined key.

Live demo: https://vtimonen.github.io/caesar-cipher/

---
## Features
- Text Encryption: Users can enter any string to be encrypted.
- Adjustable Shift: Supports a shift key ranging from 0 to 25.
- Case Sensitivity: The algorithm identifies and maintains both uppercase and lowercase letters.
- Character Preservation: Non-alphabetic characters, such as spaces and symbols, remain unchanged in the output.
